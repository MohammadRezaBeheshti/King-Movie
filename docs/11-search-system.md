# پیاده‌سازی سیستم جستجو (Search System)

## ۱. هدف مرحله

در این مرحله سیستم جستجو برای پروژه King Movie پیاده‌سازی شد.

با توجه به تعداد زیاد فیلم‌ها، سریال‌ها و انیمه‌ها، کاربران باید بتوانند به دو روش مختلف محتوا را پیدا کنند:

- جستجوی سریع از طریق Navbar
- جستجوی پیشرفته با فیلترهای مختلف

---

# ۲. معماری جستجو

سیستم جستجو به دو بخش تقسیم شد:

```
Quick Search
        |
        |
Advanced Search
```

هر بخش کاربرد متفاوتی دارد.

---

# ۳. جستجوی سریع (Quick Search)

این بخش در Navbar قرار دارد.

هدف:

پیدا کردن سریع یک عنوان خاص.

ویژگی‌ها:

- جستجو هنگام تایپ کاربر
- نمایش پیشنهادها
- محدود بودن تعداد نتایج
- انتقال مستقیم به صفحه جزئیات رسانه

جستجو بر اساس:

- عنوان فارسی
- عنوان اصلی
- عنوان انگلیسی

انجام می‌شود.

---

# ۴. جستجوی پیشرفته (Advanced Search)

این بخش برای زمانی است که کاربر دنبال محتوای خاصی است.

فیلترهای استفاده شده از مدل‌های موجود پروژه گرفته شدند.

شامل:

- نوع رسانه
- ژانر
- کشور
- سال انتشار
- امتیاز IMDb
- وضعیت انتشار
- بازیگر
- کارگردان

---

# ۵. صفحه نتایج

مسیر:

```
/search/
```

نمونه:

```
/search/?q=dark&type=series
```

صفحه نتایج شامل:

- نمایش عبارت جستجو
- نمایش فیلترهای اعمال شده
- نمایش Media Card ها
- Pagination
- حالت بدون نتیجه

است.

---

# ۶. بهینه‌سازی Query

برای جلوگیری از ایجاد Queryهای اضافی از امکانات Django ORM استفاده شد.

## Q()

برای ساخت جستجوهای ترکیبی:

مثلاً:

- عنوان فارسی شامل عبارت باشد
- یا عنوان انگلیسی شامل عبارت باشد

---

## select_related()

برای ارتباطات ForeignKey استفاده شد.

---

## prefetch_related()

برای روابط Many To Many استفاده شد.

مانند:

- ژانرها
- بازیگران
- کشورها

---

# ۷. ساختار Templateها

برای جلوگیری از ایجاد فایل‌های بزرگ، کامپوننت‌های جدا ساخته شدند.

```
components/search/

navbar-search.html

search-form.html

search-filter.html

search-result-grid.html

search-empty.html

search-pagination.html
```

---

# ۸. اصول معماری رعایت شده

- عدم وجود Query داخل Template
- استفاده از Componentهای قابل استفاده مجدد
- جداسازی منطق جستجو از رابط کاربری
- استفاده از مدل‌های موجود بدون ایجاد فیلدهای اضافی

---

# ۹. نتیجه‌گیری

با تکمیل این مرحله، کاربران می‌توانند محتوای مورد نظر خود را سریع یا با فیلترهای حرفه‌ای پیدا کنند.

این سیستم پایه قابلیت‌های آینده مانند:

- پیشنهاد فیلم
- جستجوی پیشرفته‌تر
- دسته‌بندی محتوا
- تحلیل رفتار کاربران

خواهد بود.
