# راه‌اندازی اولیه پروژه

## معرفی پروژه

پروژه King Movie یک سامانه مدیریت و معرفی فیلم، سریال و انیمیشن است که با استفاده از فریمورک Django توسعه داده می‌شود. هدف از این پروژه طراحی یک وب‌سایت کاربردی با رابط کاربری مدرن، قابلیت مدیریت محتوا، سیستم احراز هویت کاربران و امکاناتی مانند جستجوی پیشرفته، علاقه‌مندی‌ها، لیست تماشا و ثبت امتیاز و نظر است.

---

## اهداف پروژه

- طراحی یک سامانه مدیریت فیلم و سریال
- پیاده‌سازی معماری استاندارد Django
- طراحی رابط کاربری مدرن و واکنش‌گرا
- استفاده از Tailwind CSS برای توسعه سریع رابط کاربری
- استفاده از Vite جهت مدیریت Assetها
- تولید پروژه‌ای مناسب برای ارائه دانشگاهی و رزومه

---

## تکنولوژی‌های مورد استفاده

| تکنولوژی        | توضیحات                          |
| --------------- | -------------------------------- |
| Python          | زبان برنامه‌نویسی سمت سرور       |
| Django 6        | فریمورک اصلی پروژه               |
| SQLite          | پایگاه داده در مرحله توسعه       |
| Tailwind CSS v4 | طراحی رابط کاربری                |
| Vite            | مدیریت فایل‌های CSS و JavaScript |
| JavaScript      | تعاملات سمت کاربر                |
| Git & GitHub    | کنترل نسخه                       |

---

## ساختار اولیه پروژه

در ابتدای پروژه چهار اپلیکیشن اصلی ایجاد شد:

- accounts
- media_library
- interactions
- dashboard

همچنین پوشه‌های مورد نیاز برای فایل‌های Template، Static، Media و Frontend ایجاد گردید.

---

## دلیل انتخاب Django

دلایل انتخاب Django عبارت‌اند از:

- سرعت توسعه بالا
- امنیت مناسب
- وجود پنل مدیریت داخلی
- ORM قدرتمند
- ساختار استاندارد MVC (MTV)
- مناسب بودن برای پروژه‌های آموزشی و تجاری

در پایان این مرحله زیرساخت اولیه پروژه آماده گردید و بستر لازم برای توسعه بخش‌های مختلف سیستم فراهم شد.
