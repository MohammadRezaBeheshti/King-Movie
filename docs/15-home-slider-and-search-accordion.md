# بهبود اسلایدر صفحه اصلی و آکاردئون فرم جستجو

## ۱. هدف مرحله

در این مرحله تمرکز روی بهبود تجربه کاربری دو بخش مهم رابط کاربری بود:

- ردیف‌های نمایش رسانه در صفحه اصلی
- فرم جستجوی پیشرفته

هدف این بود که بخش‌های شلوغ و ایستا، به کامپوننت‌های تعاملی‌تر و جمع‌وجورتر تبدیل شوند تا در دسکتاپ، تبلت و موبایل خوانایی و پیمایش بهتری داشته باشند.

---

# ۲. تبدیل ردیف‌های صفحه اصلی به Embla Slider

قبلاً بخش‌های لیست رسانه در صفحه اصلی بیشتر به شکل نمایش گروهی/ردیفی ساده بودند. در این مرحله، کامپوننت `media-row` به اسلایدر Embla تبدیل شد.

## فایل‌های مرتبط

```
templates/pages/home.html
templates/components/media/media-row.html
frontend/js/app.js
frontend/css/app.css
```

## ساختار جدید Media Row

کامپوننت `media-row.html` اکنون شامل ساختار استاندارد Embla است:

```
media-row-slider
 └── embla__viewport
      └── embla__container
           └── embla__slide
                └── media-card
```

## رفتار Responsive

تعداد اسلایدهای قابل مشاهده بر اساس عرض صفحه تنظیم شد:

| دستگاه | تعداد اسلاید |
| ------ | ------------ |
| موبایل | ۲ |
| تبلت | ۲.۵ |
| دسکتاپ | ۵.۵ |

این رفتار با CSS Flex Basis کنترل می‌شود:

- موبایل: `flex: 0 0 50%`
- تبلت (`768px+`): `flex: 0 0 40%`
- دسکتاپ (`1024px+`): `flex: 0 0 calc(100% / 5.5)`

## مقداردهی JavaScript

در `frontend/js/app.js` تابع `initializeMediaRowSliders` اضافه شد تا تمام المان‌های دارای `data-media-row-slider` را با Embla راه‌اندازی کند.

تنظیمات مهم:

- جهت: RTL
- تراز: start
- حلقه: فقط وقتی تعداد اسلایدها از آستانه مشخص بیشتر باشد

## مزایا

- امکان اسکرول افقی روان روی کارت‌های رسانه
- نمایش نیم‌اسلاید برای نشان دادن ادامه لیست
- حفظ استفاده مجدد از `media-card`
- سازگاری بهتر با تجربه کاربری پلتفرم‌های نمایش فیلم

# ۳. تبدیل فرم جستجو به حالت آکاردئون

فرم جستجوی پیشرفته قبلاً همه فیلترها را در چند ردیف کامل نمایش می‌داد و فضای زیادی از صفحه می‌گرفت.

در این مرحله فرم به حالت جمع‌شونده تغییر کرد:

- یک ردیف اصلی همیشه نمایش داده می‌شود
- بقیه فیلترها به صورت پیش‌فرض مخفی هستند
- کاربر می‌تواند با دکمه «نمایش بیشتر» فیلترهای اضافی را باز کند

## فایل‌های مرتبط

```
templates/components/search/search-form.html
frontend/js/app.js
```

## ردیف اصلی (همیشه قابل مشاهده)

فیلترهای ردیف اول:

- جستجو
- نوع رسانه
- ژانر
- کشور

## فیلترهای اضافی (پیش‌فرض بسته)

فیلترهای مخفی:

- از سال
- تا سال
- حداقل IMDb
- وضعیت
- کارگردان
- بازیگر
- مرتب‌سازی

## دکمه کنترل

دکمه `نمایش بیشتر` اضافه شد و با کلیک:

- بخش فیلترهای اضافی باز می‌شود
- متن دکمه به `نمایش کمتر` تغییر می‌کند
- آیکون بین `+` و `−` جابه‌جا می‌شود

## منطق JavaScript

تابع `initializeSearchFormAccordion` در `app.js` اضافه شد.

رفتار فعلی:

- فرم همیشه با حالت بسته شروع می‌شود
- فقط با کلیک کاربر باز یا بسته می‌شود
- از `aria-expanded` برای دسترسی‌پذیری استفاده می‌شود

# ۴. جزئیات پیاده‌سازی Template

## Media Row

```django
<div class="media-row-slider embla mt-4" data-media-row-slider>
    <div class="embla__viewport" data-media-row-viewport>
        <div class="embla__container">
            {% for media in items %}
                <div class="embla__slide">
                    {% include "components/media/media-card.html" %}
                </div>
            {% endfor %}
        </div>
    </div>
</div>
```

## Search Form

```django
<div class="grid gap-4 sm:grid-cols-2 lg:grid-cols-4">
    <!-- فیلترهای اصلی -->
</div>

<div class="mt-4 hidden" data-search-form-extra>
    <div class="grid gap-4 sm:grid-cols-2 lg:grid-cols-4">
        <!-- فیلترهای اضافی -->
    </div>
</div>

<button type="button" data-search-form-toggle aria-expanded="false">
    نمایش بیشتر
</button>
```

# ۵. جزئیات پیاده‌سازی CSS و JS

## CSS اسلایدر ردیف رسانه

کلاس‌های جدید برای اسلایدر ردیف‌ها اضافه شد:

- `.media-row-slider`
- `.media-row-slider .embla__viewport`
- `.media-row-slider .embla__container`
- `.media-row-slider .embla__slide`

هدف این استایل‌ها:

- مخفی کردن overflow
- چیدمان افقی اسلایدها
- تعیین عرض هر اسلاید در breakpointهای مختلف
- حفظ فاصله داخلی بین کارت‌ها

## JS تعاملات UI

در این مرحله دو initializer جدید در `app.js` فعال شدند:

```js
initializeMediaRowSliders();
initializeSearchFormAccordion();
```

این توابع در کنار موارد قبلی مثل Theme و Hero Slider اجرا می‌شوند.
