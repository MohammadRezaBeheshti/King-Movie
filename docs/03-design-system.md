# سیستم طراحی (Design System)

## هدف

ایجاد یک سیستم طراحی یکپارچه برای کل پروژه تا تمامی صفحات و کامپوننت‌ها از قوانین یکسانی برای رنگ‌ها، فونت، فاصله‌ها و سایر ویژگی‌های ظاهری پیروی کنند.

---

## دلیل استفاده از CSS Variables

- پیاده‌سازی آسان Dark Mode
- مدیریت متمرکز رنگ‌ها
- افزایش قابلیت نگهداری
- جلوگیری از تکرار رنگ‌ها

---

## ساختار فایل‌های CSS

frontend/css/

- app.css
- theme.css
- fonts.css
- base.css

---

## وظیفه هر فایل

### app.css

نقطه ورود فایل‌های CSS و Import تمام استایل‌ها.

### theme.css

تعریف متغیرهای پروژه مانند رنگ‌ها، فونت‌ها و Radius.

### fonts.css

تعریف فونت‌های پروژه.

### base.css

استایل‌های پایه مانند body، html و لینک‌ها.

---

## نتیجه

با این ساختار، توسعه رابط کاربری پروژه به صورت ماژولار، قابل توسعه و قابل نگهداری انجام خواهد شد.
