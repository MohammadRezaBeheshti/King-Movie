# پیاده‌سازی داشبورد کاربر و سیستم Theme

## ۱. هدف مرحله

در این مرحله بخش کاربری پروژه King Movie توسعه داده شد.

هدف این مرحله ایجاد فضای شخصی برای کاربران و تکمیل سیستم نمایش سایت بود.

قابلیت‌های اضافه شده:

- داشبورد کاربر
- نمایش علاقه‌مندی‌ها
- نمایش لیست تماشا
- نمایش امتیازهای کاربر
- نمایش نظرات کاربر
- سیستم Dark / Light Mode

---

# ۲. داشبورد کاربر

مسیر:

```
/profile/
```

این صفحه اطلاعات و فعالیت‌های کاربر را نمایش می‌دهد.

---

# ۳. بخش‌های داشبورد

## اطلاعات کاربر

نمایش:

- نام کاربری
- ایمیل
- تاریخ عضویت

---

## علاقه‌مندی‌ها

نمایش رسانه‌هایی که کاربر به Favorite اضافه کرده است.

داده‌ها از مدل:

```
Favorite
```

دریافت می‌شوند.

---

## لیست تماشا

نمایش محتواهایی که کاربر برای مشاهده آینده ذخیره کرده است.

داده‌ها از مدل:

```
Watchlist
```

استفاده می‌کنند.

---

## امتیازها

نمایش امتیازهایی که کاربر برای رسانه‌ها ثبت کرده است.

داده‌ها از:

```
Rating
```

دریافت می‌شوند.

---

## نظرات

نمایش و مدیریت نظرات ثبت شده توسط کاربر.

داده‌ها از:

```
Review
```

استفاده می‌کنند.

---

# ۴. ساختار Template

برای جلوگیری از ایجاد فایل‌های بزرگ، داشبورد به کامپوننت‌های مختلف تقسیم شد.

ساختار:

```
components/profile/

profile-header.html

favorite-list.html

watchlist-list.html

rating-list.html

review-list.html
```

---

# ۵. سیستم Theme

برای مدیریت حالت روشن و تاریک سایت، سیستم Theme پیاده‌سازی شد.

ویژگی‌ها:

- Dark Mode
- Light Mode
- ذخیره انتخاب کاربر
- هماهنگی با تنظیمات سیستم کاربر

---

# ۶. روش پیاده‌سازی Theme

تم با استفاده از:

- CSS Variables
- JavaScript
- LocalStorage

مدیریت می‌شود.

نمونه:

```
:root
```

برای حالت روشن و:

```
.dark
```

برای حالت تاریک استفاده می‌شود.

---

# ۷. هماهنگی کامپوننت‌ها

سیستم Theme در تمام بخش‌های پروژه هماهنگ شد:

- Navbar
- Home
- Media Detail
- Search
- Profile
- Cards
- Forms

---

# ۸. اصول معماری رعایت شده

در این مرحله:

- رنگ‌ها Hard Code نشده‌اند.
- کامپوننت‌ها از متغیرهای Theme استفاده می‌کنند.
- Templateها فقط مسئول نمایش هستند.
- منطق Theme از UI جدا شده است.

---

# ۹. نتیجه‌گیری

با تکمیل این مرحله، کاربران دارای فضای شخصی شده‌اند و رابط کاربری پروژه قابلیت تغییر تم دارد.

King Movie اکنون دارای زیرساخت‌های اصلی یک پلتفرم نمایش محتوای مدرن است:

- مدیریت محتوا
- تعامل کاربران
- جستجو
- پروفایل
- سیستم Theme
