# مدیریت حرفه‌ای محتوا و Django Admin

## ۱. هدف مرحله

در این مرحله پنل مدیریت Django برای پروژه King Movie توسعه داده شد.

هدف این مرحله تبدیل Admin پیش‌فرض Django به یک پنل مناسب برای مدیریت محتوای سینمایی بود.

قابلیت‌های اضافه شده:

- مدیریت بهتر Media
- نمایش پیش‌نمایش تصاویر
- مدیریت فصل و قسمت سریال‌ها
- بهبود مدیریت عوامل و دسته‌بندی‌ها
- ایجاد داده آزمایشی

---

# ۲. بهبود Media Admin

پنل Media شامل امکانات زیر شد:

## نمایش اطلاعات مهم

در لیست Media:

- عنوان فارسی
- نوع رسانه
- وضعیت انتشار
- امتیاز IMDb
- تاریخ انتشار

نمایش داده می‌شود.

---

## جستجو

امکان جستجو بر اساس:

- عنوان فارسی
- عنوان اصلی
- عنوان انگلیسی
- Slug
- بازیگران

اضافه شد.

---

## فیلترها

فیلترهای زیر اضافه شدند:

- نوع رسانه
- وضعیت
- ژانر
- تاریخ انتشار

---

## Preview تصاویر

برای مدیریت راحت‌تر محتوا:

- پوستر
- تصویر پس‌زمینه

در Admin قابل مشاهده هستند.

---

# ۳. مدیریت فصل و قسمت

برای رسانه‌های سریالی ساختار زیر ایجاد شد:

```
Media
 |
 └── Season
        |
        └── Episode
```

در Admin:

- فصل‌ها داخل صفحه سریال مدیریت می‌شوند.
- قسمت‌ها داخل فصل مربوطه مدیریت می‌شوند.
- فیلم‌ها بخش فصل و قسمت ندارند.

---

پنل مدیریت موارد زیر توسعه داده شد:

امکانات:

- Search

# ۵. سیستم داده آزمایشی

برای تست سریع پروژه یک Django Management Command ایجاد شد.

مسیر:

```
media_library/management/commands/seed_sample_media.py
```

اجرا:

```bash
python manage.py seed_sample_media
```

است.

---

# ۷. Idempotent بودن Seed

دستور Seed طوری طراحی شده که اجرای چندباره آن باعث ایجاد داده تکراری نمی‌شود.

مثال:

```
python manage.py seed_sample_media
python manage.py seed_sample_media
```

اجرای دوم داده‌ها را دوباره ایجاد نمی‌کند.
